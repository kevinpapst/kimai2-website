Kimai - time-tracking for everyone

Kimai is a free open source timetracker. It tracks work time and prints out a summary of your activities on demand. Yearly, monthly, daily, by customer, by project, by action … It’s simplicity is its strength. Due to Kimai’s web browser based interface it runs cross-platform.

While there are many programs one can use for time and project tracking, they tend to be overkill for my rather simple needs. Kimai came about because I was tired to feed lame boring excel tables …

Kimai does not have to run while recording, you can even quit your browser. Your time is still recording until you stop it from any web browser that has access to your installation. It is designed to hold lots of users, but you can also use it for a single user if you want to.

Kimai can be localized to any language, though we only ship three sets of languages files in the new version for now: English, German and Russian. Feel free to send your self-made language files – we’re looking for translators and would appreciate your support!

Feature list

The following list is not complete, but it should give you a quick overview of all the features that Kimai v2 offers.

Name Description Free version
Multi-user Kimai v2 can be used by as many users as you want: each installation supports unlimited users with different roles (see permissions)
Print invoices Different templates, calculator, invoice number generator and you can add your own templates
Manage customers Manage your customers and set address, contacts, country, currency, visibility (for archiving) and many more fields
Manage projects Manage projects for each customer with order number, budget and visibility (for archiving)
Manage activities Activities are used to record timesheet entries. Manage activities for each project with order number, budget and visibility (for archiving)
Archiving After a long usage a lot of entries can pile up in your dropdowns - now you can hide customers, projects and activities
Duration only If your local laws require you to only show durations instead of start and end date-time, you can activate that mode globally
Round times Global config for rounding of time-records, with different settings for each weekday: e.g. configure that Kimai always rounds up to 15 minutes on weekdays and 1 hour at weekends (can be deactivated)
Weekend rates You can configure global settings for rate factors on single weekdays, e.g. 150% for saturday and sunday
Statistics Your dashboard will show you several statistics about recorded times and the amount of collected data. We can add more charts, just ask us!
Multiple records You can record as many times as you want in parallel, a limit can be configured
Permissions Assign different roles to your user: User can record times, teamleads can print invoices and configure project/activities, admins can configure users and customers (read more)
Filter times You can filter all times by user, project, activity and date
Different themes Kimai bundles different color themes and display settings, so every user can tweak it to his personal flavour
User profile Personal statistics and per user settings for hourly rate, Kimai theme, password and avatar
Extendable Kimai v2 was built with extendability in mind, it is extendable with Symfony bundles and has many extension points
Modern technologies Based on newest technologies and frameworks: PHP7, Symfony 4, Doctrine, Flex, Bootstrap, Webpack and many more...
SQLite support No database setup required, Kimai has SQLite support out-of-the-box. But you want can use MySQL/MariaDB as well.
